EclipseCoder language support

Identifier:
net.fornwall.eclipsecoder.languagesupport

Since:
0.1

Description:
Extension point to register support for additional programming languages.

Configuration Markup:

<!ELEMENT extension (languagesupport)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT languagesupport EMPTY>

<!ATTLIST languagesupport

class    CDATA #REQUIRED

language CDATA #REQUIRED>

  • class - Language support class implementing fornwall.net.eclipsecoder.languagesupport.LanguageSupport
  • language - The programming language being supported by this plugin

Examples:
From the Java language support plugin: 
<extension point=
"net.fornwall.eclipsecoder.languagesupport"
>


<languagesupport
 language=
"Java"

 class=
"net.fornwall.eclipsecoder.javasupport.JavaLanguageSupport"
/>


</extension>
